教你怎么免费搭建hexo博客

发表于 2020-04-16 | 分类于 技术 ， hexo

1. 安装 执行前先安装nodejs

随便找一个目录 打开控制台(cmd)

2. 创建并运行hexo

3. 修改 _config.yml

修改标题 作者等信息

4. 安装主题（可选）

这边推荐next

5. 新建文章

然后hexo就会在 source/_posts 下面 新建一个 markdom 文件 今天天气真好.md

新建后直接刷新浏览器如果看不到效果

需要ctrl+c停止 hexo serve

然后再执行 一遍 就好了

6. 部署服务器

1. 首先需要注册一个码云账号 https://gitee.com/

2. 新建一个仓库

开源->勾选上公开

3. 修改_config.yml

4. 执行命令上传编译后的代码到仓库

5. 开启码云的 gitee pages服务 部署访问

点击部署 骚等片刻 你的博客就部署好了

访问地址格式为
https://<码云英文账户名>.gitee.io/<博客项目名>/

# 随笔 # hexo
微服务gateway增加文件上传大小限制
